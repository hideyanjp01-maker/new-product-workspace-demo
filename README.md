# 新品工作台 - 交互式演示Demo

这是一个完整的新品工作台演示系统，支持完整的工作流程和交互功能。系统包含18个角色工作台，每个角色都有对应的设计图片，100%还原设计稿。

## 功能特性

✅ **18个角色工作台** - 包含新品灵感洞察器、AI洞察、市场分析、品牌负责人、商品管理、电商负责人、天猫运营、天猫流量运营、京东运营、京东流量运营、抖音店铺、抖音投放、抖音内容制作、抖音商务、潜力达人洞察、竞争品VS、素材生成器、任务管理等  
✅ **100%图片还原** - 使用设计好的图片作为页面背景，完全还原设计稿  
✅ **完整工作流** - 18步完整工作流程，从新品洞察到任务管理  
✅ **角色指标关联** - 每个角色都有对应的业务指标展示  
✅ **交互功能** - 支持点击、状态流转、工作流导航等完整交互  
✅ **模拟数据** - 包含产品信息、角色数据、指标数据等丰富的模拟数据

## 使用说明

### 页面导航

- **首页** (`/`) - 显示统计信息、最近产品和所有角色工作台入口
- **产品列表** (`/products`) - 查看所有产品，支持筛选和搜索
- **产品详情** (`/products/:id`) - 查看单个产品的详细信息和工作流历史
- **工作流** (`/workflow`) - 执行完整的18步工作流程
- **角色页面** (`/roles/:roleId`) - 查看各个角色工作台，100%还原设计图片

### 完整工作流程（18步）

1. **新品灵感洞察** - 发现新品灵感和市场机会
2. **AI洞察分析** - AI驱动的智能洞察分析
3. **市场分析** - 市场趋势和竞争分析
4. **品牌策略** - 品牌策略和定位管理
5. **商品管理** - 商品信息和管理
6. **电商整体规划** - 电商整体运营管理
7. **天猫运营** - 天猫平台运营管理
8. **天猫流量运营** - 天猫流量获取和优化
9. **京东运营** - 京东平台运营管理
10. **京东流量运营** - 京东流量获取和优化
11. **抖音店铺运营** - 抖音店铺运营管理
12. **抖音投放** - 抖音广告投放管理
13. **抖音内容制作** - 抖音内容创作和管理
14. **抖音商务合作** - 抖音商务合作管理
15. **潜力达人洞察** - 达人分析和合作机会
16. **竞品分析** - 竞品对比分析
17. **素材生成** - 营销素材生成工具
18. **任务管理** - 任务管理和分配

### 工作流操作

- **点击步骤** - 在工作流页面点击任意步骤，自动跳转到对应的角色页面
- **查看设计** - 在角色页面可以看到100%还原的设计图片
- **完成步骤** - 在角色页面点击"完成此步骤"按钮，自动进入下一步
- **查看指标** - 每个角色页面都会显示相关的业务指标
- **返回工作流** - 随时可以返回工作流页面查看整体进度

## 自定义配置

### 修改模拟数据

编辑 `src/data/mockData.ts` 文件，可以：
- 添加更多产品数据
- 修改产品信息
- 添加工作流历史记录

### 修改工作流步骤

编辑 `src/store/workflowStore.ts` 文件，可以：
- 修改工作流步骤
- 调整步骤顺序
- 添加新的步骤

### 角色和指标

系统定义了18个角色，每个角色都有：
- 对应的设计图片（100%还原）
- 相关的业务指标
- 工作流步骤关联

指标包括：
- 销售额、流量、转化率
- 品牌知名度、内容播放量
- 达人合作数等

所有角色和指标的关联关系都在 `src/data/mockData.ts` 中定义。

## 技术实现

- **图片展示** - 使用 `RolePage` 组件展示设计图片，支持100%还原
- **工作流管理** - 使用 Zustand 管理工作流状态，支持步骤跳转和状态更新
- **路由导航** - 使用 React Router 实现页面路由和角色页面导航
- **指标关联** - 每个角色都有对应的指标数据，动态展示

## 常见问题

### Q: 如何添加新的页面？
A: 在 `src/pages/` 目录下创建新组件，然后在 `src/App.tsx` 中添加路由。

### Q: 如何修改工作流步骤？
A: 编辑 `src/store/workflowStore.ts` 中的 `initialSteps` 数组。

### Q: 图片不显示怎么办？
A: 确保图片文件在 `public/images/` 目录下，并且路径正确。在代码中使用 `/images/文件名` 的路径。
